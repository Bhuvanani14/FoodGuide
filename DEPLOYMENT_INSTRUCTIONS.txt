## 🚀 QUICK DEPLOYMENT (5 Minutes)

### Prerequisites
1. ✅ gcloud CLI installed
2. ✅ Authenticated: `gcloud auth login`
3. ⏳ **Billing ENABLED** (this is required!)

### Step 1: Enable Billing
```
1. Go to: https://console.cloud.google.com
2. Select project: tidy-node-494313-j1
3. Click Billing in left menu
4. Link a billing account
5. Wait 2-3 minutes for activation
```

### Step 2: Deploy
```bash
cd /path/to/FoodGuide
chmod +x deploy-cloud-run.sh
./deploy-cloud-run.sh
```

### Step 3: Get Your URL
The script will output something like:
```
🌐 Service URL: https://foodguide-ai-meal-planner-abc123xyz.run.app
```

✅ **You're live!**
